┌─────────────────────────────────────────────────────────────────────┐
│  ❌ COMMON MISTAKES                                                  │
└─────────────────────────────────────────────────────────────────────┘

"taskResponseScore": "6.5"       ❌ String → ✅ 6.5 (number)
"taskResponseScore": 6.75        ❌ Wrong step → ✅ 6.5 or 7.0
"category": "grammar"            ❌ Lowercase → ✅ "GRAMMAR"
"severity": "MEDIUM"             ❌ Uppercase → ✅ "medium"
endIndex: 10, startIndex: 20     ❌ Wrong order → ✅ Must be >
"explanation": "Wrong verb"      ❌ English → ✅ Vietnamese

Nested structure:                ❌ WRONG FORMAT
{
  "scores": { ... },             ← Don't nest!
  "feedback": { ... }            ← Must be flat!
}

┌─────────────────────────────────────────────────────────────────────┐
│  🎓 SEVERITY GUIDELINES                                              │
└─────────────────────────────────────────────────────────────────────┘

high:
  • Subject-verb agreement errors
  • Wrong tense usage
  • Spelling of common words
  • Not answering the question

medium:
  • Preposition errors
  • Article errors (a/an/the)
  • Word choice issues
  • Inappropriate linking words

low:
  • Simple word instead of academic
  • Minor punctuation
  • Word repetition
  • Informal expressions

┌─────────────────────────────────────────────────────────────────────┐
│  📝 FEEDBACK TIPS (VIETNAMESE)                                       │
└─────────────────────────────────────────────────────────────────────┘

generalFeedback (100-200 words):
  → Nhận xét tổng quan
  → Đề cập yêu cầu đề bài
  → Điểm mạnh và yếu chính

strengths (50-100 words):
  → Liệt kê cụ thể
  → Ví dụ: "Sử dụng linking words tốt như 'Moreover'"
  → Ví dụ: "Từ vựng phong phú: 'mitigate', 'substantial'"

improvements (50-100 words):
  → Gợi ý cải thiện cụ thể
  → Ví dụ: "Cần luyện câu điều kiện loại 2"
  → Ví dụ: "Tránh lặp từ 'many', dùng 'numerous'"

┌─────────────────────────────────────────────────────────────────────┐
│  🚨 TROUBLESHOOTING                                                  │
└─────────────────────────────────────────────────────────────────────┘

Problem: Gemini trả về markdown ```json { ... } ```
Solution: Nhắc "Chỉ trả về pure JSON, không markdown"

Problem: Scores là string "6.5" thay vì number
Solution: Nhắc "Tất cả scores phải là NUMBER không phải string"

Problem: category là "grammar" (lowercase)
Solution: Nhắc "category phải UPPERCASE: GRAMMAR"

Problem: severity là "MEDIUM" (uppercase)
Solution: Nhắc "severity phải lowercase: medium"

Problem: Gemini trả về nested {scores: {}, feedback: {}}
Solution: Dùng transformNestedToFlat() trong test-ai-output.js

┌─────────────────────────────────────────────────────────────────────┐
│  💡 PRO TIPS                                                         │
└─────────────────────────────────────────────────────────────────────┘

1. Test với nhiều loại bài viết:
   • Band 5.0 (nhiều lỗi)
   • Band 6.5 (trung bình)
   • Band 8.0 (tốt)

2. Kiểm tra edge cases:
   • Bài viết không có lỗi (errors = [])
   • Bài viết rất ngắn (< 150 words)
   • Bài viết off-topic

3. Fine-tune prompt nếu cần:
   • Thêm examples cụ thể
   • Nhấn mạnh format rules
   • Adjust feedback tone

4. Monitor output quality:
   • Check explanation có giá trị không
   • Check startIndex/endIndex accuracy
   • Check feedback relevance
